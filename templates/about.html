About This Project

This project, "Sanskrit Data Stats," is developed by CIKS, IITR. It provides tools and visualizations for analyzing Sanskrit data.

Our goal is to offer insights into various aspects of the Sanskrit language through data-driven approaches.

The project is open-source and the code is available on GitHub.
